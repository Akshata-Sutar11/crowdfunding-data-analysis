# crowdfunding-data-analysis
Data analysis project exploring crowdfunding campaign success using SQL, Excel and Tableau

📊 Crowdfunding Campaign Data Analysis

📌 Project Overview

This project analyzes crowdfunding campaign data to identify factors that influence the success or failure of fundraising campaigns. The analysis focuses on campaign categories, funding goals, duration, and backer engagement to uncover meaningful insights.
The goal of this project is to demonstrate data analysis, data cleaning, SQL querying, and data visualization skills using multiple analytical tools.

🎯 Project Objectives

Analyze crowdfunding campaign performance.

Identify key factors affecting campaign success.

Explore trends across different campaign categories.

Provide insights that can help improve campaign strategy.

🛠 Tools & Technologies Used

SQL – Data querying and analysis

Microsoft Excel – Data cleaning and preprocessing

Tableau – Data visualization and trend analysis

  🔍 Key Business Questions

What percentage of crowdfunding campaigns are successful?

Which campaign categories receive the highest funding?

How does campaign duration affect success rates?

What role do backers play in campaign performance?

📈 Key Insights

Certain campaign categories consistently attract higher funding.

Campaigns with moderate funding goals tend to have higher success rates.

Projects with more backers and engagement show higher chances of reaching their funding goals.

Campaign duration impacts campaign visibility and success probability.

💡 Business Recommendations

Focus marketing efforts on high-performing categories.

Set realistic funding goals to increase campaign success probability.

Encourage early backer engagement to build campaign momentum.

Optimize campaign duration for better visibility and participation.
